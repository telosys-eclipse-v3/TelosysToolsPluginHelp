Templates Language

The code generation is based on "templates"
A template is a file with a ".vm" extension written in "Velocity Template Language"
Velocity is a well known Open Source template engine ( cf http://velocity.apache.org/ )

You need a template file for each artifact you want to generate ( Java class, HTML file, etc... )
You can update existing templates or create new templates according to your needs.

The Velocity engine used in Telosys Tools is version 1.7
The main aspects of the Velocity Templates Language (VTL) are documented here :

  • #assertFalse
  • #assertTrue
  • #error
  • #foreach
  • #if
  • #include
  • #macro
  • #parse
  • #set
  • #using
  • comments
  • unparsed content
